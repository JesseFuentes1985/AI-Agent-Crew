## Mission Control Dashboard — Built Today

- Files: `workspace-main/dashboard/server.js`, `dashboard/index.html`
- Data source: `workspace-main/agent-tasks.json`

### Features
- Home grid — 8 agent cards with real avatars, skills, progress bars, task lists
- Agent detail pages — full kanban board (Queue / In Progress / Done) with drag-and-drop
- Inline task edit (✏️) and delete (🗑️) on kanban cards
- Checkbox-only toggle on home cards (not full row click)
- Notes system — saves to both `agent-tasks.json` AND each agent's `workspace-{agent}/NOTES.md`

- Skills editor per agent — add/remove, saves to both JSON and TASKS.md
- Credit balance widget (top right) — manual entry, warns red under $5, saves to `dashboard/budget.json`
- localStorage caching for instant load, background fetch for freshness
- Optimistic checkbox toggle (no lag)
- Filter tabs: All / Has Pending / Completed
