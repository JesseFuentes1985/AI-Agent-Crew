# Skill: Visual Creation

Translating ideas into visuals — for AI image generation, Blender, and concept art.

## Concept Art Direction
Describe visuals so precisely that an AI image generator or human artist can execute them.

### Prompt Formula
`[Subject] + [Style] + [Lighting] + [Mood] + [Camera/Angle] + [Details]`

**Example — The Hex:**
> A weathered bone-colored freighter/gunship hybrid drifting in deep space. Hard surface sci-fi style, cinematic lighting with amber running lights and blue emissive engine glow. Battle-scarred hull with "SECTOR 7X" in red stencil on the port side, "HEX" hand-stenciled in green on starboard. Low angle shot emphasizing width and mass. Dark, lived-in, dangerous energy.

### Style References
- Hard SF: Mass Effect, The Expanse, Alien
- Space Opera: Star Wars, Guardians of the Galaxy
- Dark/Gritty: Event Horizon, Blade Runner
- Painterly: Ralph McQuarrie concept art, John Harris

## 3D Model Ideation
Translate ideas into Blender-ready concept specs.

### Before Opening Blender
1. Sketch the silhouette — complex detail means nothing if the shape doesn't read
2. Define the scale reference (what human-sized object is nearby?)
3. Identify the 3 most important visual details
4. Decide the polygon budget — hero asset or background piece?

### Blender Workflow for Ships
1. Block out basic shape with primitives
2. Add major panel lines and surface breaks
3. Add functional details (engines, weapons, landing gear)
4. Add wear and damage
5. UV unwrap + texture (PBR: diffuse, roughness, metallic, normal)
6. Light for the scene

## Ship Design
Starships, stations, and megastructures.

### Ship Design Philosophy
- Form follows function — what does this ship DO? Let that define its shape
- Silhouette first — a great ship reads as a clear shape at thumbnail size
- Scale sells reality — tiny windows, massive engines, visible wear
- Every ship tells a story — who built it, for what purpose, how long ago?

### Ship Categories for Last Star
| Class | Role | Size | Design Feel |
|---|---|---|---|
| Fighter/Interceptor | Fast attack | Small | Aggressive, angular |
| Freighter | Cargo/utility | Medium | Functional, modular |
| Gunship | Combat/escort | Medium | Armored, purposeful |
| Cruiser | Fleet backbone | Large | Military, imposing |
| Carrier | Ship deployment | Massive | Industrial, utilitarian |
| Dreadnought | Capital ship | Enormous | Ancient, overwhelming |
| Station | Fixed installation | Variable | Layered, lived-in |

## Environment Design
Cities, planets, and landscapes.

### Environment Pillars
1. **Ecology** — what grows here? What survives?
2. **History** — what was built, and what was destroyed?
3. **Economy** — what do people do here to survive?
4. **Atmosphere** — what does it FEEL like to stand here?

### Palette-Driven Design
Choose 3 colors that define the environment before adding detail:
- Neon/wet asphalt/fog = cyberpunk megacity
- Bone/rust/sand = dead civilization
- Deep blue/bioluminescence/black = alien ocean world

## Symbol & Faction Emblems
Logos and insignias that carry meaning.

### Symbol Design Rules
1. Readable at small size — if it doesn't work as a thumbnail, redesign
2. Geometric but not generic — avoid basic triangles/circles with no modification
3. Encode meaning — a faction's symbol should tell you something about them
4. Consider negative space — what's absent says as much as what's present

### Process
1. Describe the faction's core value in one word
2. Find 3 real-world symbols/shapes associated with that word
3. Combine and simplify
4. Test: can you draw it from memory after seeing it once?
