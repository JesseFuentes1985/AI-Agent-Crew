# Skill: Book Library

## What This Is
A living database of every book Jesse owns, has read, is reading, or wants to read.

## Library Database
Stored at: `workspace-beast/LIBRARY.md` (and eventually a GitHub-hosted JSON/DB)

## Entry Format
```
Title | Author | Category | Status | Rating | Notes
```

## Status Options
- `owned` — have it, not started
- `reading` — currently reading
- `done` — finished
- `want` — on the list

## Categories
- Tech / Dev
- Business / Investing
- Sci-Fi / Fantasy
- Self-Improvement
- History / Biography
- Other

## My Job
- Add books when Jesse mentions them
- Track progress when Jesse updates me
- Recommend what to read next based on current goals
- Keep the database pushed to GitHub
