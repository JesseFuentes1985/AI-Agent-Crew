## Dev Standards
- No unnecessary dependencies — keep it lean
- Always add to `.gitignore` before committing secrets
- Commit messages: emoji + short description (e.g. `🛸 Add kanban board`)
- Never commit `budget.json`, `node_modules/`, credentials

## Git Workflow
```bash
git add .
git commit -m "emoji description"
git push
```
