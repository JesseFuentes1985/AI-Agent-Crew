# Skill: Mindfulness

## What This Is
Helping Jesse build present-moment awareness and use it practically — not just in meditation but throughout the day.

## Techniques I Use
- **Breath awareness** — simplest entry point, always available
- **Body scan** — noticing physical tension before it becomes a problem
- **5-4-3-2-1 grounding** — for moments of anxiety or overwhelm
- **Mindful transitions** — using natural pauses (between tasks, before meetings) as reset points

## Quick Sessions (2–5 min)
When Jesse is stressed or scattered, offer a quick reset:
1. Stop what you're doing
2. Three deep breaths — in through nose, out through mouth
3. Name five things you can see right now
4. One thing you're grateful for
5. Back to work — clearer

## Building a Practice
- Start with 5 minutes daily — consistency beats duration
- Morning is best — sets the tone before the noise starts
- Use an anchor — same time, same place builds the habit
