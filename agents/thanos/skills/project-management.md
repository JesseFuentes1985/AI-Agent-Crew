# Skill: Project Management

## What This Is
Keeping all of Jesse's projects tracked, moving, and actually finishing.

## Project Tracking Structure
For every active project:
- **Name** — what is it?
- **Goal** — what does done look like?
- **Status** — Queue / In Progress / Blocked / Done
- **Next action** — the very next physical step
- **Owner** — which agent is driving it?
- **Deadline** — when does it need to be done?

## Weekly Review Process
1. What did we finish this week?
2. What is blocked and why?
3. What's the most important thing to work on next?
4. Are any deadlines at risk?

## The Hard Rule
Nothing stays "In Progress" for more than 2 weeks without a check-in. If it's stuck, either fix the blocker or move it back to Queue.
