# Skill: Fitness

## What This Is
Building and tracking workout plans for Jesse — practical, achievable, consistent.

## Approach
- Start where Jesse is, not where he should be
- Consistency > intensity
- Mix of strength, cardio, and mobility based on goals
- Account for recovery — overtraining is real

## Workout Planning
1. Ask about current fitness level, equipment available, time per week
2. Build a plan that fits his schedule
3. Track progress in memory files
4. Adjust based on feedback

## Recovery
- Rest days are part of the plan
- Sleep and nutrition fuel performance
- If Jesse is sore or injured → scale back and recommend rest
